# NexoPOS - Sistema de Gestión de Kiosco

NexoPOS es un sistema de punto de venta (POS) diseñado para facilitar la gestión de kioscos y pequeños negocios.

## Características principales
- **Gestión de Ventas:** Interfaz rápida con buscador de productos.
- **Control de Inventario:** Validación de stock en tiempo real.
- **Cierre de Turno:** Arqueo de caja y generación automática de tickets.
- **Multiusuario:** Gestión de roles (Admin/Empleado).
